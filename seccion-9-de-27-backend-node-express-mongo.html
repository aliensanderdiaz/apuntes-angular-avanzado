Sección 95 de 343 - NOTA - Error conexión mongoDB (NETWORK ACCESS)

En la siguiente clase realizamos la configuración de Mongo DB Atlas, pero si a la hora de acceder a nuestra base de datos desde MongoDB Compass o desde el backend obtenemos un error de acceso, puede ser debido a la configuración de "Network Access".
Para solventar esto, vamos a MongoDB Atlas, y una vez logueados, en el menú izquierdo seleccionamos "Network Access".
En esta ventana clickamos al botón verde + ADD IP ADDRESS.
Tenemos dos opciones:

  • Podemos darle a ADD CURRENT IP ADDRESS para que añada nuestra dirección IP y tener acceso desde ahí.
  • Podemos añadir la IP 0.0.0.0/0, con lo que abriríamos nuestra BBDD para que cualquier dirección IP pueda conectarse.
Una vez le demos a "Confirm", esperamos unos minutos para que se aplique el cambio y ya deberíamos de poder acceder.
